# GitHub – First push

1. Create repo at https://github.com/new (name e.g. `WATS`). Do **not** add README/.gitignore.
2. Push: `git remote add origin https://github.com/USER/WATS.git` then `git push -u origin main`.
3. Auth: use Personal Access Token (Settings → Developer settings → Tokens) as password, or switch to SSH.

Credentials and secrets stay out of the repo (use `.env` and GitHub Secrets).
